================================================
SOMA — GitHub Pages デプロイ手順
================================================

【必要なファイル（GaitAnalysisフォルダに置く）】
  soma_camera.html    ← 今回更新版（PWA対応）
  patient_report.html
  research_report.html
  manifest.json       ← 今回新規
  sw.js               ← 今回新規（Service Worker）
  soma_icon.png       ← 任意（アプリアイコン画像）

================================================
STEP 1: GitHubアカウント作成
================================================
https://github.com → Sign up（無料）
メールアドレスで登録

================================================
STEP 2: リポジトリ作成
================================================
1. github.com にログイン
2. 右上「+」→「New repository」
3. 設定：
   Repository name: soma-gait
   ✅ Public（必須）
   ✅ Add a README file
4. 「Create repository」をクリック

================================================
STEP 3: ファイルをアップロード
================================================
1. リポジトリページで「Add file」→「Upload files」
2. 以下のファイルをドラッグ&ドロップ：
   - soma_camera.html
   - patient_report.html
   - research_report.html
   - manifest.json
   - sw.js
3. 「Commit changes」をクリック

================================================
STEP 4: GitHub Pages を有効化
================================================
1. リポジトリの「Settings」タブ
2. 左メニュー「Pages」
3. Source: 「Deploy from a branch」
4. Branch: 「main」/ 「/ (root)」を選択
5. 「Save」

数分後にURLが発行される：
https://[GitHubユーザー名].github.io/soma-gait/

================================================
STEP 5: iPhoneでPWAとしてインストール
================================================
1. iPhoneのSafariで以下を開く：
   https://[ユーザー名].github.io/soma-gait/soma_camera.html

2. 画面下部の「共有」ボタン（□↑）をタップ

3. 「ホーム画面に追加」をタップ

4. 名前を「SOMA」として「追加」

5. ホーム画面にSOMAアイコンが出現 → タップで起動！

================================================
完成後のURL構成
================================================
計測ツール:
  https://[ID].github.io/soma-gait/soma_camera.html

患者レポート:
  https://[ID].github.io/soma-gait/patient_report.html

研究者レポート:
  https://[ID].github.io/soma-gait/research_report.html

================================================
注意事項
================================================
・GitHubアカウントは無料で作成可能
・Public リポジトリはファイルが公開されます
  患者の個人情報は入れないでください
  （このツールはアルゴリズムのみで個人情報なし）
・更新方法：GitHubでファイルを上書きアップロードするだけ
・オフライン動作：初回アクセス後はキャッシュで動作
